Sphere

  • Create a Sphere with two clicks

Tool Operation

  1. (Optional) Key-in the number of segments for the sphere to set the smoothness.
  2. Click a point to set the center of the sphere.
  3. Click a point to set the radius of the sphere.

Tips

  • When setting the radius of the sphere, you can key-in an exact measurement instead of clicking a point.
  • Use the arrow keys to lock the orientation of the sphere to the red, green, or blue axis.
