Random Position

  • Randomizes the x-y-z position of the objects.

Tool Operation

  1. (Optional) Preselect groups and components.
  2. Click a group or component.
  3. Click to set the red axis jitter or key-in a value.
  4. Click to set the green axis jitter or key-in a value.
  5. Click to set the blue axis jitter or key-in a value.
  6. Continue clicking to apply a random position to each selected object.

Tips

  • Use the SketchUp Axes Tool to modify the orientation of the red, green, and blue axes.
