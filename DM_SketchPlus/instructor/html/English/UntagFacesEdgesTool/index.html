Untag Faces and Edges

  • Recursively removes Tags from only Faces and Edges within the selected entities

Tool Operation

  1. (Optional) Preselect entities to remove tags from.
  2. Click an entity and all of the tags applied to the faces and edges will be removed from it as well as from all nested faces and edges within.

Tips

  • Tags applied to groups and components will remain.
  • Remember that edges and faces that are untagged will inherit the tag of the object that contains them.
  • Press ESC to clear the selection.
