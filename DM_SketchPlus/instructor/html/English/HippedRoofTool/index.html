Draw Hipped Roof

  • Generate a hipped roof from the selected horizontal face.

Tool Operation

  1. (Optional) Preselect a horizontal face.
  2. Click a horizontal face to generate a hipped roof.
  3. Click again to define the slope and height of the hipped roof.

Tips

  • Key-in an exact slope in degrees instead of clicking a height reference point.
  • You can also key-in a slope expressed as RISE:RUN (for example 5:12).
  • Double-click to apply the last used slope to a new horizontal face.
  • This tool always uses the global model Z direction regardless of the direction of the current sketch axes.
