Walk+

  • Precision first-person camera control using arrow keys

Tool Operation

  1. Use arrow keys to walk.
  2. Click and drag to look around.
  3. Use the '+' or '-' adjust the walking speed.

Modifier Keys

  • CTRL = Hold to run
  • ALT = Toggle walking through walls

Tips

  • Key-in eye height to adjust camera height above walking surface.
