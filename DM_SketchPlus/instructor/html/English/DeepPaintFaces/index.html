Deep Paint Faces

  • Apply materials directly to faces even if nested deep within a group or component

Tool Operation

  1. Click any face in your model to apply the currently selected material.

Modifier Keys

  • ALT = Sample and select a material from the model.
  • SHIFT = Paint all faces that match the material of the clicked face that are at the same entities level as the clicked face.
  • CTRL = Paint all faces with the same material as the clicked face that are adjacent to the clicked face.

Tips

  • Remember: This tool can modify faces that are deep inside nested groups and components.
