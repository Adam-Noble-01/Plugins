Untag Groups and Components

  • Recursively removes Tags from only Groups and Components within the selected entities

Tool Operation

  1. (Optional) Preselect groups and components to remove tags from.
  2. Click a group or component and its tags will be removed as well as from all nested groups and components within.

Tips

  • Tags applied to edges and faces will remain.
  • Remember that edges and faces that are untagged will inherit the tag of the object that contains them.
  • Press ESC to clear the selection.
