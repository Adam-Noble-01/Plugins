Cone

  • Create a Cone with three clicks

Tool Operation

  1. (Optional) Key-in the number of segments for the cone to set the smoothness.
  2. Click a point to define the center and orientation of the base.
  3. Click a point to define to radius of the base.
  4. Click a point to define the height of the cone.

Tips

  • Use the arrow keys to lock the orientation of the cone to an axis.
  • When setting the radius and height, you can key-in an exact measurement instead of clicking a point.
