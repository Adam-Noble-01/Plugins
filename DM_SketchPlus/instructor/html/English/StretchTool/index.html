Deep Stretch

  • Select and move entities inside nested objects

Tool Operation

  1. Click and drag to window-select entities to stretch.
  2. Click the first reference point for the stretch.
  3. Click the target point to set the stretch direction and distance.
  4. (Optional) Key-in a distance during or after a stretch.

Modifier Keys

  • SHIFT = Lock inference when stretching

Tips

  • Use arrow keys to lock stretch to an axis direction.
  • Arrow key locking can be done prior to picking the first reference point.
  • Press ESC to cancel while performing a stretch.
  • Press ESC again to clear the selection.
