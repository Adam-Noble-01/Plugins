Draw Pipes

  • Create pipes by clicking points or pre-selecting a series of edges

Tool Operation

  1. (Optional) Select a pre-existing path of edges for the pipe to follow.
  2. Enter the pipe diameter and bend radius in the measurements box separated by a comma.
  3. Start a new pipe by clicking on an existing pipe or any point in your model.
  4. Continue adding points to the pipe path. While adding points, you can enter the length and bend radius in the measurements box.

Modifier Keys

  • SHIFT = Hold to lock inference
  • ESC = Stop drawing a pipe and reset the tool

Tips

  • You can modify the diameter and bend radius of an entire pipe immediately after creation.
  • Use arrow keys to lock to the red, green, or blue axes.
  • Change the number of segments of the pipe profile by entering a number followed by 's'.
