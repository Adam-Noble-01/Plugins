Smart Array

  • Create multiple copies of an object based on the relative position, scale, and rotation of the first copy

Tool Operation

  1. (Optional) Preselect a component.
  2. Click a component to select it as the source object to copy.
  3. Click another instance of the same component to set the relative transformation of the copies in the array.
  4. Move the mouse to set the number of copies or key-in a value.

Tips

  • Press ESC to reset the tool and pick a new source component.
  • If you use the tool on a group, it will be converted to a component automatically.
