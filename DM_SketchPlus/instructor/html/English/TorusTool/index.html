Torus

  • Create a Torus with three clicks

Tool Operation

  1. (Optional) Key-in the number of segments for the torus to set the smoothness.
  2. Click a point to define the center and orientation of the torus.
  3. Click a point to define to radius of the torus.
  4. Click a point to define the radius of the torus profile.

Tips

  • Use the arrow keys to lock the orientation of the torus to an axis.
  • When setting the radius, you can key-in an exact measurement instead of clicking a point.
