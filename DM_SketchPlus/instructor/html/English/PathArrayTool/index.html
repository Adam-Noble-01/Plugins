Path Array

  • Copy a component along the pre-selected path

Tool Operation

  1. Preselect connected edges or a single face to define the path.
  2. Activate the tool.
  3. Move the cursor near the desired start point of the path and click to confirm.
  4. Click a component in your model to copy along the selected path.
  5. Move the mouse to set the spacing of the copies along the path or key-in a precise value.
  6. Click to confirm the spacing value and generate the copies.
  7. Click another component and repeat the steps if desired.

Modifier Keys

  • CTRL = Toggle max spacing or fixed spacing.
  • SHIFT = Toggle rotate instances to align with path direction.
  • ALT = Reverse the path direction

Tips

  • Press Esc to reset the tool and reselect the start point for the path.
